Sketch-Board

Instructions:

Move your mouse over sketchboard to draw in the sketchboard

    Variations:

  • Draw using One color: On Mouse Move over the grid On Mouse Click over the grid
  • Draw using random color: On Mouse Move over the grid On Mouse Click over the grid
  • Incremental color change: Fadeout Effect on Mouse Out Incremental Color on Mouse Click
  • Trail: Trail Effect on Mouse Move
    • Please Select any One Option from above


Enter a Grid Size value between 1 and 128.
